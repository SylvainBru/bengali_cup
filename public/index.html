📖 Règles et Légende du classement

⚽ Attribution des points

  • Victoire : 3 points
  • Match Nul : 1 point
  • Défaite : 0 point

En cas d'égalité, le départage se fait sur la différence de buts, puis sur les buts marqués.

📊 Légende du tableau

Pts : Points totaux
J : Matchs joués
V : Victoires
N : Matchs nuls
D : Défaites
BP : Buts pour (marqués)
BC : Buts contre (encaissés)
Diff : Différence de buts
